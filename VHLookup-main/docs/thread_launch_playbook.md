# Threads Launch Playbook

목표 독자는 공공기관, 교육청, 학교, HR, 총무, 예산 담당자입니다. 메시지는 “AI가 대신 판단한다”가 아니라 “내 PC에서 원본을 건드리지 않고 반복 엑셀 업무를 끝낸다”에 맞춥니다.

## Core Hook

엑셀 VLOOKUP 때문에 매번 야근하는 행정직에게:

> 파일만 넣으면 제출자료 수합, 명단 대조, 누락자 확인, 오류 리포트까지 로컬 PC에서 끝납니다. AI도 클라우드도 안 씁니다.

## Post Ideas

1. 공공기관 행정직이 제일 싫어하는 엑셀 업무

   여러 부서에서 온 파일을 하나로 합치는데, 누구는 1행에 제목을 쓰고 누구는 3행부터 헤더를 씁니다. 이 도구는 실제 헤더를 찾고 표준 컬럼으로 맞춘 뒤 통합본과 확인 리포트를 만듭니다.

2. VLOOKUP을 잘 모르는 사람도 쓰는 명단 대조

   기준명단과 제출명단을 고르면 `사번`과 `직원번호`처럼 이름이 다른 키 컬럼도 자동 추천합니다. 부서/직급/소속을 붙이고 `기준표에 없음`, `중복 키`, `형식 불일치`를 한글로 표시합니다.

3. 보안 담당자에게 설명하기 쉬운 엑셀 자동화

   AI 없음. API 없음. 클라우드 업로드 없음. 선택한 로컬 파일만 읽고 원본 파일은 수정하지 않습니다. 보안 검토용 설명서도 같이 제공합니다.

4. 행정직에게 중요한 건 예쁜 대시보드보다 바로 보이는 결과

   결과 파일 첫 시트는 실제 업무 결과입니다. 확인할 점은 `확인사항` 시트 한 장에 모았습니다.

5. 자동으로 맞췄으면, 무엇을 맞췄는지도 보여줘야 합니다

   결과 파일의 `확인사항` 시트에서 `사번`과 `직원번호`를 어떻게 연결했는지, 어떤 컬럼을 가져올 값으로 봤는지 확인할 수 있습니다.

6. 00123 때문에 매칭 실패한 적 있다면

   `00123`과 `123`처럼 앞자리 0이 빠져 생기는 형식 불일치를 별도 리포트로 분리합니다. 몰래 붙이지 않고 확인하게 합니다.

7. 월별 가로표를 다시 쓸 수 있는 자료로

   `1월, 2월, 3월`로 옆으로 긴 표를 `기관명, 항목, 월, 값` 형태로 바꿔 수합과 대조에 다시 쓸 수 있게 합니다.

## Demo Flow

1. `run_demo.bat` 더블클릭
2. `demo_output\01_제출자료_수합결과.xlsx`로 정상 수합 보여주기
3. `demo_output\02_오류검증_제출자료.xlsx`에서 `확인사항` 시트 보여주기
4. `demo_output\03_교육이수_명단대조.xlsx`에서 `확인사항` 시트의 자동 매칭 근거 보여주기
5. `demo_output\05_수당예산_대조결과.xlsx`에서 복합 키 대조 보여주기
6. `demo_output\01_제출자료_수합결과.xlsx`에서 `확인사항` 시트의 개인정보 의심 요약 보여주기
7. “원본 파일을 수정하지 않았다”는 점을 강조하기
8. `run_cli_examples.bat`으로 GUI 설치 전에도 실제 파일 경로 기반 실행이 가능하다는 점 보여주기
9. `cli_output\00_inspect_cli.xlsx`로 사전점검 후 실행하는 안전한 흐름 보여주기

## Words To Avoid

- AI 자동 판단
- 클라우드 동기화
- 기관별 시스템 연동
- 매크로 배포
- 검증되지 않은 시간 단축률

## Words To Use

- 로컬 PC
- 원본 미수정
- 확인사항
- 제출자료 수합
- 누락자 확인
- 기준표 대조
- 공공기관 보안 검토에 설명 가능한 구조
