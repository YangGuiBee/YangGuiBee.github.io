# 2026-07-04 업데이트 내역

## 열 합치기와 값 붙이기

- 컬럼명이 달라도 동의어, 이름 유사도, 실제 값 겹침을 함께 사용해 키 컬럼을 추천합니다.
- `사번`과 `직원번호`, `관리번호`와 `접수ID`처럼 이름이 달라도 값이 충분히 겹치면 같은 키 후보로 봅니다.
- `00123`과 `123`처럼 앞자리 0 표시가 다른 숫자형 키를 같은 대상으로 비교합니다.
- 월별 지급자료처럼 단일 열만으로 중복되는 경우 `사번 + 지급월` 같은 복합 키를 더 우선 추천합니다.
- 자동 추천이 틀릴 때는 기존처럼 `컬럼 매칭 수정`에서 드롭박스로 직접 지정할 수 있습니다.
- 어느 한쪽 파일에만 있는 기준값도 삭제하지 않고 결과에 남깁니다.
- 한쪽 파일에만 있는 행이나 열은 빈 칸에 노란색과 메모를 남겨 확인할 수 있게 했습니다.
- 결과 엑셀의 보조 설명은 여러 시트로 나누지 않고 `확인사항` 시트 한 장에 모았습니다.
- 합치기 화면 기본값을 `자동 선택`으로 바꿔, 같은 양식은 행 합치기, 기준표/신청자료 조합은 열 합치기로 추천합니다.
- 합치기 결과 첫 시트에서 `원본 파일명`, `원본 시트명`, `원본 행 번호` 같은 내부 추적 컬럼을 제거했습니다.
- 파일 합치기 결과에서는 개인정보 의심 컬럼을 따로 점검하거나 색칠하지 않습니다.

## 저장 방식

- 주요 작업은 실행 시 저장 창을 띄워 사용자가 결과 파일 폴더와 파일명을 직접 선택합니다.
- 저장 창에서 취소하면 결과 파일을 만들지 않습니다.

## 전/후 파일 검증

- 전 파일과 후 파일의 컬럼명이 바뀌어도 실제 값 겹침이 높으면 같은 컬럼으로 비교합니다.
- 기준값의 앞자리 0 표시가 달라도 같은 행으로 비교합니다.
- 후 파일에서 빠진 행과 새로 추가된 행은 `확인사항` 시트에서 함께 확인합니다.
- 변경된 셀은 첫 시트 `후파일_메모`에 색과 메모로 표시합니다.
- 후 파일에 새로 생긴 행은 첫 시트에서 파란색 전체 행으로 표시합니다.
- 후 파일에서 사라진 행은 첫 시트 맨 아래에 추가하고 빨간색 전체 행으로 표시합니다.

## 피벗 요약표

- 평균, 합계, 최대, 최소 결과 숫자를 보기 좋게 정리했습니다.
- 긴 소수는 최대 소수 2자리로 표시합니다.
- 정수는 `100.0` 대신 `100`처럼 표시합니다.
- 큰 숫자는 `1,000`처럼 천 단위 콤마로 표시합니다.
- 엑셀 결과 셀은 문자열이 아니라 숫자 값으로 유지합니다.
